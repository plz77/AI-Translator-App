# AI-Translator-App
A simple Android application that utilizes Python for processing; a project showcase for my class.


Description
    An AI-powered voice translation application built using MIT App Inventor as the mobile interface and a Python API hosted on Google Colab as the translation engine.
    The app supports multi-language translation with integrated Text-to-Speech (TTS) functionality.


Technologies Used
  - Mobile Framework: MIT App Inventor.
  - Language Engine: Deep Translator (Python).
  - Web Framework: Flask & Ngrok (for local-to-public tunneling).
  - Communication: REST API (JSON).


How to Use
  - Run the Backend: Open the Google Colab script and run all cells to start the Python server.
  - Get the URL: Copy the public URL generated by Ngrok (e.g., https://...ngrok-free.dev).
  - Update App Inventor: In the Blocks editor of MIT App Inventor, paste that URL into the Web1.Url block.
  - Translate: Open the app on your phone, type your text, choose a destination language, and tap translate!.
